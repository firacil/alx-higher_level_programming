Test Cases for 1-my_list module
--------------------------

1-my_list module will print sorted list and inherits from list


Usage
======

Improting function from module:
	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()

::

	>>> print(my_list)
	[]

::

	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> print(my_list)
	[1]

::

	>>> my_list.append(4)
	>>> print(my_list)
	[1, 4]

::

	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]

::

	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

::

	>>> print(my_list)
	[1, 4, 2, 3, 5]

::

	>>> my_list.append('L')
	>>> my_list.print_sorted()
	Traceback (most recent call last):
		...
	TypeError: '<' not supported between instances of 'str' and 'int'

::

	>>> type(my_list)
	<class '1-my_list.MyList'>

If list passed

::

	>>> x = [2, 1, 3]
	>>> my_list = MyList(x)
	>>> my_list.print_sorted()
